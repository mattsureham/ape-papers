## Discovery
- **Idea selected:** idea_1029 — Japan's 2020 Equal Pay Act, zero published evaluations of a reform affecting 37% of workforce
- **Data source:** MHLW Basic Survey on Wage Structure — Excel files downloadable directly, but format changed across years (xls→xlsx, table renumbering). e-Stat API timed out repeatedly.
- **Key risk:** COVID-19 confound (treatment coincides with pandemic), aggregate data with few effective clusters

## Execution
- **What worked:** The staggered firm-size rollout provides a credible quasi-experimental design. The "compression, not uplift" finding is genuinely surprising and narratively compelling. Extending data back to 2014 gave 6 pre-periods.
- **What didn't:** Only 3 firm-size categories = 3 effective clusters. Aggregate published tables can't distinguish compositional shifts from within-firm changes. The industry continuous-treatment specification showed null effects.
- **Review feedback adopted:** Moderated causal language throughout (from "find" to "observed pattern suggests"). Added explicit discussion of composition vs within-firm wage adjustments. Strengthened COVID limitations section. Acknowledged that MHLW wages exclude bonuses/allowances (the reform's key margin).
