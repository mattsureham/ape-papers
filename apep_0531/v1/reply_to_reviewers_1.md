# Reply to Reviewers

## Reviewer 1 (GPT-5.2) — Reject and Resubmit

**1.1 Identification needs an IV or shift-share design.**
We acknowledge this is a limitation of the TWFE approach. However, we note that (a) the continuous dose-response design with 41 clusters provides substantial variation, (b) the event study confirms flat pre-trends, (c) the null result is conservative — omitted variable bias would need to precisely offset a true PCSO effect to produce zero. We discuss this in Section 4.3 (Threats to Validity). A shift-share design using pre-2010 grant dependence is an excellent suggestion for future work but beyond the scope of the current data.

**1.2 Controlling for sworn officers is a potential bad control.**
We now clarify in the text that Columns 1 and 2 of Table 2 represent different estimands (total effect vs. direct effect holding officers fixed) and that neither is automatically preferred. The stability of the coefficient across both specifications is itself informative.

**1.3 Only 2 pre-treatment years.**
This is a data constraint. Crime data in the current format begins in 2007/08 (mapped to 2008). We acknowledge the limitation in the text.
