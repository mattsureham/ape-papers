# Research Plan: apep_0710

## Research Question

Does war erode procurement institutions? Specifically, does the competitive advantage generated by Ukraine's ProZorro UAH 200,000 mandatory open-auction threshold — which generates 10-17% price savings in peacetime — collapse after the February 2022 full-scale Russian invasion, and is this erosion concentrated in frontline oblasts?

## Identification Strategy

**Two-part design:**

1. **Peacetime RD (2017–2021):** Sharp RD at the UAH 200,000 threshold. Running variable = estimated contract value. Outcomes: price savings rate, number of valid bids, probability of competitive procedure. The threshold is set by law and not easily manipulable — contract values are estimated by the procuring entity before publication. McCrary density test verifies no bunching.

2. **Wartime Diff-in-Disc (2022–2024):** Difference-in-discontinuities: compare the RD estimate pre-invasion vs post-invasion. This identifies whether the threshold-induced competition premium changed after martial law (Cabinet Resolution No. 169, Feb 24, 2022). Geographic heterogeneity: frontline vs rear oblasts.

**Primary specification (Part 1):**
Y_i = f(V_i - c) + β * Above_i + ε_i

Where V_i is contract value, c = 200,000 UAH, Above_i = 1(V_i > c). Local polynomial regression using rdrobust.

**Primary specification (Part 2 — Diff-in-Disc):**
Y_i = f(V_i - c) + β1 * Above_i + β2 * Post_i + β3 * Above_i × Post_i + ε_i

β3 identifies the wartime change in the threshold effect.

**Triple-difference extension:**
Y_i = ... + β7 * Above_i × Post_i × Frontline_i + ...

β7 identifies differential erosion in frontline oblasts.

## Expected Effects

- **Peacetime RD:** Above-threshold tenders should show higher price savings (10-17%), more bidders (~1-2 additional), and nearly 100% competitive procedure compliance.
- **Wartime erosion:** The competition premium should decline after Feb 2022 as emergency decrees allow direct contracting. Effect should be larger in frontline oblasts (Donetsk, Luhansk, Zaporizhzhia, Kherson, Kharkiv).
- **Placebo:** Remote western oblasts (Lviv, Ivano-Frankivsk, Zakarpattia) should show stable RD throughout.

## Data Source and Fetch Strategy

**Primary:** ProZorro public API (public.api.openprocurement.org/api/2.5/tenders)
- Paginate feed to collect tender IDs across 2017-2024
- Fetch individual tender details in parallel batches
- Extract: value.amount, award value, procurementMethodType, procuringEntity.address.region, date, bids count, status
- Filter to tenders with expected value 50K-500K UAH (captures threshold neighborhood)
- Target: ~15,000-25,000 tenders for RD analysis

**Frontline classification:** Based on Ukrainian government's recognized combat zones. Binary: oblast was part of active combat zone at any point during 2022-2024.

## Key Risks

1. **Manipulation:** Procuring entities may strategically set values just below 200K to avoid competitive requirements. McCrary test is essential.
2. **Threshold changes:** The UAH 200K threshold may have been adjusted for inflation. Need to verify it remained fixed throughout 2017-2024.
3. **Sample size near cutoff:** RD requires sufficient observations in narrow bandwidth around cutoff.
4. **Currency effects:** UAH depreciated significantly during war. Real vs nominal threshold matters.
