## Discovery
- **Idea selected:** idea_1781 — Panama Canal expansion (June 2016) as trade infrastructure shock
- **Data source:** Census QWI API — reliable, fast, county-level quarterly employment by NAICS
- **Key risk:** Small cluster count (26 counties, 4 controls). Addressed with expanded port list.
- **Idea pivots:** Vietnam GSO data inaccessible; 4 other ideas overlapped with existing APEP papers

## Execution
- **What worked:** Census QWI API provided clean, complete data. The surprise finding (negative effect on East/Gulf ports) was more interesting than the expected positive effect. Pre-trends were perfectly clean.
- **What didn't:** AIS vessel tracking data was in the idea manifest but not used — would strengthen the continuous treatment intensity design. Binary DiD is coarser than ideal.
- **Review feedback adopted:** Added acknowledgment of AIS limitation, wild cluster bootstrap note, tempered "null result" framing to reflect the significant negative finding.
