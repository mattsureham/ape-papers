# Reply to Reviewers — Round 1

## Internal Review (CC)

1. **Gilets jaunes speculation** — Softened the language in the distributional consequences subsection. Removed the causal framing linking the protests to the housing subsidy reform, replacing with a more measured observation about the 2024 reinstatement.

2. **Control group weighting** — The commune-year fixed effects already absorb level differences. Population-weighting would shift identification toward larger communes, which is a different estimand. The current specification gives equal weight to within-commune variation, appropriate for the policy question.

3. **Commune-year aggregation** — Acknowledged as a limitation. Transaction-level analysis for 2021-2024 would require harmonizing with the CDC aggregates for 2014-2020, introducing a measurement discontinuity. The current approach is consistent across the full panel.

All other points noted as minor; no changes needed.
