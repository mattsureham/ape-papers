## Discovery
- **Idea selected:** idea_2537 — NZ MDRS upzoning composition test, only 43% overlap with nearest APEP paper
- **Data source:** Stats NZ Building Consents Excel — reliable download, but dwelling type only at region level (not TA)
- **Key risk:** Small treated-unit count (4 regions) and construction downturn dominating the signal

## Execution
- **What worked:** Clean pre-trends (p=0.39), precise null finding, multiple estimators agree
- **What didn't:** Could not get TA × dwelling type data from the Excel release — forced region-level analysis
- **Review feedback adopted:** Added attenuation bias discussion, MDE calculation, fixed placebo framing
- **Key lesson:** Stats NZ Infoshare/microdata portal would give monthly TA × dwelling type data, but requires a different data access pathway. Future NZ housing papers should start there.
