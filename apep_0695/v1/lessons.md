## Discovery
- **Idea selected:** idea_0037 — Mass denationalization in the Dominican Republic (TC/0168). Selected for uniqueness: no parallel in economics literature.
- **Data source:** ILO SDMX API + World Bank WDI API + DHS API. ENI microdata unavailable (requires ONE institutional registration).
- **Key risk:** Province-level DiD impossible with national-level outcomes. Pivoted to national ITS.

## Execution
- **What worked:** The "statistical invisibility" framing turned a data limitation into the finding. Power calculation showing MDE >> predicted effect made the null interpretable.
- **What didn't:** Province-level DiD — WDI/ILO data has no cross-sectional variation. DHS has subnational data but only pre-2013 surveys.
- **Review feedback adopted:** Strengthened power calculation with better baseline formality assumptions; clarified ENI access barriers; sharpened contribution vs Amuedo-Dorantes (2017).
