# Internal Review — Round 1

**Date:** 2026-03-05
**Paper:** The Price of Pork: France's Dual-Mandate Ban and the Fiscal Cost of Local-National Connections

## Verdict: MINOR REVISION

### Strengths
1. Clean identification strategy with 248 treated / 291 control constituencies
2. Comprehensive null result across 6 fiscal outcomes with tight confidence intervals
3. Event study confirms parallel trends over 10 pre-treatment years
4. Strong institutional background section, especially the 2017 turnover discussion
5. HonestDiD sensitivity analysis and triple-difference add depth

### Issues Requiring Revision

**Methodology / Design**
- No fatal issues identified. Advisor review passed 4/4.

**Exhibits**
- Table 3: variable name "treated_placebo" should be cleaned up for publication
- Figure A4 (département bar chart): nearly unreadable with 90+ bars — should be simplified or removed
- Consider promoting triple-diff table to main text (most interesting heterogeneity result)

**Prose**
- Opening paragraph is a civics lesson rather than a hook — needs stronger opening
- Results section reads like table narration ("Column 1 shows...") — needs narrative energy
- "0.014 thousand euros" is clunky — ground results in real-world magnitudes (14 euros per person)
- Data section spends too many paragraphs on harmonization mechanics

**Minor**
- Abstract uses "precisely estimated null effects" — could be stronger as "a clear zero"
- Some "Threats to Identification" bullet points could be woven into prose

### Recommended Changes
1. Rewrite opening paragraph with a hook
2. Rewrite first paragraph of Results 5.1 to lead with the economic story, not the table
3. Add "14 euros per person" translation when first reporting the coefficient
4. Clean up Table 3 variable labels
5. Trim data harmonization section, move some detail to appendix
